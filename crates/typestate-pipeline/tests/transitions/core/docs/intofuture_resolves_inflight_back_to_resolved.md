## InFlight `IntoFuture`

**Invariant.** An InFlight carrier directly `.await`s into the
corresponding Resolved carrier of the same state. This is the
machinery `pipelined!` / `impl_pipelined!` emit so users can `.await`
on a chain's terminal step without having to remember which mode
they're in.

**Failure mode this guards.** The carrier's `IntoFuture` impl is
generated by `impl_pipelined!`. If it broke, every chain would fail
to compile at the terminal `.await`. The test pins the trivial
single-step case so a regression surfaces here without needing a
full chain to reproduce.

**Setup.** Start `Registered`, call async-deferred `tag_version(3)`
producing `Author<Versioned, InFlight>`, then directly `.await` it.

**Assertion.** Yields `Author<Versioned, Resolved>` with state
`version == 3` and `manifest_hash == 9`.
